================================================================================
                                1. PENDAHULUAN
================================================================================

Aplikasi BreastLens adalah sistem deteksi dini kanker payudara berbasis kecerdasan 
buatan yang dikembangkan untuk membantu skrining awal. Aplikasi ini menggunakan 
teknologi deep learning dengan arsitektur Convolutional Neural Network (CNN) 
untuk menganalisis gambar mammogram dan memberikan prediksi kemungkinan adanya 
kanker payudara.

Tujuan utama aplikasi ini adalah memberikan akses mudah dan cepat untuk deteksi 
dini, sehingga dapat membantu meningkatkan tingkat kesembuhan melalui penanganan 
yang lebih awal.

================================================================================
                              3. CARA MENJALANKAN APLIKASI
================================================================================

LANGKAH 1: MENJALANKAN BACKEND SERVER
-------------------------------------
1. Buka terminal/command prompt
2. Navigate ke direktori project: cd D:\coding\breastcancer_novalidator
3. Aktifkan virtual environment (jika ada): Scripts\activate
4. Install dependencies: pip install -r requirements.txt
5. Jalankan server: python app.py
6. Server akan berjalan di: http://localhost:5000

LANGKAH 2: MENJALANKAN FLUTTER APP
----------------------------------
1. Buka terminal/command prompt
2. Navigate ke direktori Flutter: cd D:\coding\breastcancer_novalidator\breast_cancer_flutter_app
3. Install dependencies: flutter pub get
4. Jalankan aplikasi:
   - Web: flutter run -d chrome
   - Android: flutter run -d android
   - iOS: flutter run -d ios

LANGKAH 3: VERIFIKASI KONEKSI
-----------------------------
1. Buka aplikasi Flutter
2. Klik tombol "Test Koneksi" di bagian bawah
3. Pastikan muncul pesan "Server terhubung!"

A. MENU UTAMA (HOME)
--------------------
1. Upload Gambar:
   - Klik tombol "Galeri" untuk memilih dari galeri foto
   - Klik tombol "Kamera" untuk mengambil foto baru
   - Pastikan gambar jelas dan tidak blur

2. Analisis Gambar:
   - Setelah gambar terpilih, klik tombol "Analisis"
   - Tunggu proses analisis selesai (akan muncul loading)
   - Hasil akan ditampilkan dengan detail lengkap

3. Hasil Analisis:
   - Jika Non-Kanker: Pesan selamat dan saran BreastLens
   - Jika Kanker: Peringatan dan panduan langkah selanjutnya
   - Tingkat kepercayaan ditampilkan dalam persentase

---------------
1. Melihat Riwayat:
   - Klik tab "Riwayat" di bottom navigation

2. Detail Analisis:
   - Klik pada item riwayat untuk melihat detail
   - Popup akan menampilkan:
     * Gambar yang dianalisis
     * Hasil prediksi lengkap
     * Tanggal dan waktu analisis
     * Tingkat kepercayaan

3. Hapus Riwayat:
   - Klik tombol delete (üóëÔ∏è) pada item riwayat
   - Konfirmasi penghapusan

C. MENU BreastLens
--------------
1. Panduan BreastLens:
   - Klik tab "BreastLens" di bottom navigation
   - Baca langkah-langkah BreastLens yang tersedia
   - Ikuti panduan visual yang diberikan

2. Pengingat Bulanan:

   - Klik "Setel Pengingat BreastLens Bulanan"

3. Edukasi Kanker Payudara:
   - Scroll ke bagian "Pelajari tentang kanker"

                                 6. TROUBLESHOOTING
================================================================================

MASALAH UMUM DAN SOLUSI:

1. SERVER TIDAK TERHUBUNG
   Gejala: Pesan "Server tidak terhubung"
   Solusi:
   - Pastikan Flask server berjalan di terminal
   - Cek URL server: http://localhost:5000
   - Restart server jika perlu

2. GAMBAR TIDAK TERUPLOAD
   Gejala: Error saat memilih gambar
   Solusi:
   - Pastikan izin kamera/galeri sudah diberikan
   - Coba gambar dengan ukuran lebih kecil
   - Restart aplikasi

3. ANALISIS GAGAL
   Gejala: Error saat analisis
   Solusi:
   - Pastikan server Flask berjalan
   - Cek koneksi internet
   - Coba dengan gambar yang berbeda

4. RIWAYAT TIDAK MUNCUL
   Gejala: Riwayat kosong atau tidak update
   Solusi:
   - Refresh halaman riwayat
   - Restart aplikasi
   - Cek koneksi server

5. TEMA TIDAK BERUBAH
   Gejala: Toggle tema tidak berfungsi
   Solusi:
   - Restart aplikasi
   - Cek provider state management
